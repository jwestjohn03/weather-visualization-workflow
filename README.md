# Automated Weather Visualization Workflow

This project uses the OpenWeather API to build an automated workflow for collecting and visualizing historical U.S. weather data.

The pipeline compares two user-selected years and generates choropleth maps for:
- Temperature (°F)
- Wind speed (mph)
- Humidity (%)

It also includes difference maps to highlight changes between the two years.

## Output
Sample visualizations are included in the `visuals/` folder.

## Tools Used
- Python
- pandas
- plotly
- OpenWeather API
